# Homepage featured selection — vial-only images

Drop **final** PNGs here **by hand**: transparent background, vial only. Filename must match the shop asset basename (e.g. `bpc-157-clean-2.png`).

These files are **not** generated by `npm run images:match-backgrounds` (that script only adjusts backgrounds on the main `public/products/*.png` files used on the shop).

If a matching file is missing here, the homepage featured block uses the regular shop image instead.
